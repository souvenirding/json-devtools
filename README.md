# JSON DevTools

A lightweight, browser-based JSON DevTools for developers.

👉 Live Demo: https://souvenirding.github.io/json-devtools/

---

## ✨ Features

- 📦 JSON Formatter (pretty print)
- 🧹 JSON Minifier
- 🔍 JSON Validator with error hinting
- 🌲 Tree-style JSON viewer (collapsible)
- 🔎 Search inside JSON
- ✏️ Real-time editing with auto refresh
- 📋 One-click copy result
- ⚡ Fully client-side (no backend, no data upload)

---

## 🚀 Why this tool?

When working with APIs, logs, or backend responses, you often need a quick way to:

- Inspect JSON structure
- Debug malformed responses
- Format messy logs
- Compare or locate fields quickly

This tool is designed to be:

> Fast • Simple • Privacy-first • Dev-friendly

No login. No server. Just open and use.

---

## 🧠 Use Cases

### API Debugging
Paste API response and instantly inspect structure.

### Backend Logs
Format nested JSON logs for readability.

### Payment / Webhook Debugging
Useful for Stripe / PayPal / webhook payload inspection.

### Data Analysis
Quickly explore nested JSON data without writing scripts.
